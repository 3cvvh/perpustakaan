## ðŸš€ Cara Instalasi & Menjalankan
1. **Clone repositori ini:**
   ```bash
   git clone https://github.com/3cvvh/perpustakaan.git
   ```
2. **Masuk ke folder proyek:**
   ```bash
   cd perpustakaan
   ```
3. **Jalankan server lokal (misal XAMPP):**
   - Pastikan folder ini berada di dalam direktori `htdocs` XAMPP.
   - Aktifkan Apache & MySQL melalui XAMPP Control Panel.
4. **Import database:**
   - Buka `phpMyAdmin`.
   - Import file database yang tersedia (misal: `perpustakaan.sql`).

5. **Akses aplikasi:**
   - Buka browser dan akses: [http://localhost/tes/perpustakaan](http://localhost/tes/perpustakaan)
